## 云端进度上报

`POST /api/test-executions/{id}/report`

请求体：

```json
{
  "status": "running",
  "errorMessage": "",
  "conversation": [],
  "executionLog": "[{\"stage\":\"generating\",\"message\":\"Agent正在处理\",\"detail\":[{\"time\":\"20260408-15:40:22\",\"message\":\"任务已发送\"}]}]"
}
```

说明：

- `executionLog` 传字符串
- 字符串内容是阶段日志数组

阶段值：

- `pending`
- `preparing`
- `generating`
- `validating`
- `constraint_scoring`
- `static_scoring`
- `completed`

## 本地文件

每个 Agent 目录下有三类文件：

- `*_opencode_sse_full.jsonl`
- `*_opencode_sse_events.jsonl`
- `*_opencode_progress_events.jsonl`

其中：

- `full` 用于保存当前 session 的完整原始 SSE，便于排查问题
- `events` 用于保存裁剪后的原始事件
- `progress` 用于本地页面展示
