# 鸿蒙开发 Skill 评测用例（100条）

> **前提约束**
> - 状态管理：统一使用 V2 装饰器（`@ObservedV2`、`@Trace`、`@ComponentV2`、`@Local`、`@Param`、`@Provider`、`@Consumer`、`@Monitor`、`@Computed`）
> - 路由管理：统一使用 `Navigation` + 路由表，禁止使用旧版 `router`
> - 覆盖范围：每类用例兼顾**普通应用**与**元服务**两种形态
